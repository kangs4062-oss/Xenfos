How to run locally:
1. Save the files and run a local server in this folder:
   - Python 3: python -m http.server 8000
   - Node: npx http-server -p 8000
2. Open http://localhost:8000/3d-logo-generator-pro.html in a modern browser.
3. For best reflections, replace the HDR URL in the HTML with your own CORS-enabled HDR.
